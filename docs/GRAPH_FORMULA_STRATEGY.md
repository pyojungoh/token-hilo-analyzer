# 그래프만 보고 승을 끌어올리는 공식 방향

그래프는 **맨 왼쪽이 최신 회차**이고, 같은 결과가 이어지면 세로로 쌓인다(한 열 = 한 연승/연패 구간).  
연패가 많고 손실이 나는 구간에서 승률을 끌어올리려면 아래 방식을 공식에 반영하면 된다.

---

## 1. 그래프에서 읽는 정보

| 정보 | 의미 |
|------|------|
| **맨 왼쪽 열의 색** | 직전 결과가 정(초록)인지 꺽(빨강)인지 |
| **맨 왼쪽 열의 높이** | 현재 **연속 같은 결과의 길이** = 연승/연패 길이 |
| **긴 빨간 열이 여러 개** | 꺽(연패)가 길게 이어진 구간이 반복됨 |

즉, “연패가 많고 돈을 잃는” 구간 = **빨간 열이 길고 많은** 구간이다.

---

## 2. 공식 쪽에서 할 수 있는 것 (줄 vs 퐁당)

- **줄(유지)**: 직전과 **같은** 결과 예측 (직전 꺽 → 다음도 꺽).
- **퐁당(바뀜)**: 직전과 **반대** 결과 예측 (직전 꺽 → 다음은 정).

연패가 많다 = “꺽이 계속 나온다” = 줄(유지) 쪽 예측이 계속 실패했다는 뜻이다.  
그래프만 봤을 때 승을 끌어올리려면:

1. **연패가 길어지면 “바뀜” 쪽을 더 믿기**  
   - 현재 **꺽(연패) run 길이**가 N 이상이면 → **퐁당(바뀜) 가중치를 올린다.**  
   - “연패가 곧 끊길 가능성”을 반영해 **다음은 정** 쪽으로 픽을 내는 방식.

2. **이미 쓰고 있는 패턴 보정 유지**  
   - **V자 패턴** (긴 줄 → 퐁당 1~2 → 짧은 줄): 연패가 많으므로 퐁당 가중치 +0.12.  
   - **U자 + 줄 3~5**: 연패가 많으므로 줄 가중치 +0.10 (유지 쪽으로 한 번 넘기기).

3. **연패 길이만 따로 보정** (추가)  
   - **현재 run이 꺽(연패)이고 길이 ≥ 4**이면 → 퐁당 가중치 추가 상향 (예: +0.14), 줄 가중치 하향.  
   - 그래프에서 “긴 빨간 열”이 나왔을 때, **그 다음 회차는 정이 나올 가능성을 높게 보는** 식.

---

## 3. 정리: 어떤 방식으로 공식을 짜야 하는가

- **그래프만 봤을 때**  
  - “맨 왼쪽 열 = 최신 run”  
  - “그 열이 빨간(꺽)이고 **높이가 크다**” → 연패 길이가 길다.  
- **공식 쪽**  
  - **연패 길이(현재 꺽 run 길이)가 길수록** → **퐁당(바뀜) 가중치를 더 올린다.**  
  - 그러면 “다음은 정” 쪽으로 픽이 나와, 연패 구간에서 승을 끌어올리는 효과를 노릴 수 있다.

즉, **그래프의 “현재 열 높이(연패 길이)”를 수치로 넣어서, 길수록 퐁당 쪽을 강하게 주는 방식**으로 공식을 짜면 된다.
